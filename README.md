# Семейный бюджет — Telegram Bot

## Быстрый старт

### 1. Получить токен бота
1. Открыть Telegram, найти @BotFather
2. Написать /newbot
3. Придумать имя и username боту
4. Скопировать токен вида `7123456789:AAHxx...`

### 2. Установить зависимости
```bash
pip install python-telegram-bot==20.7
```

### 3. Вставить токен
В файле bot.py найти строку:
```python
TOKEN = os.environ.get("BOT_TOKEN", "ВСТАВЬТЕ_ВАШ_ТОКЕН_ЗДЕСЬ")
```
Вставить токен прямо туда, или задать переменную окружения:
```bash
export BOT_TOKEN="7123456789:AAHxx..."
```

### 4. Запустить
```bash
python bot.py
```

## Как пользоваться ботом

### Добавить расход
Просто напишите сумму и категорию:
```
850 продукты Пятёрочка
3500 красота стрижка
авто 2000 бензин
2500 детское Детский мир
1200 здоровье аптека
жкх 9500 квартплата
```

### Добавить доход
```
+95000 зарплата
доход 15000 фриланс
+1800 кэшбэк
```

### Команды
| Команда | Что делает |
|---|---|
| /отчёт | Сводка за текущий месяц с графиком категорий |
| /история | Последние 10 операций |
| /история 20 | Последние 20 операций |
| /категории | Список всех категорий |
| /удалить 42 | Удалить операцию #42 |

### Исправить категорию
После каждого расхода бот показывает кнопки:
- **✅ Верно** — сохранить как есть
- **✏️ Изменить категорию** — выбрать другую из списка

## Размещение на сервере (чтобы бот работал 24/7)

### Вариант A: Railway (бесплатно)
1. Создать аккаунт на railway.app
2. Нажать "New Project" → "Deploy from GitHub"
3. Залить папку budget_bot на GitHub
4. В Railway добавить переменную окружения BOT_TOKEN
5. Всё — бот запустится автоматически

### Вариант B: Render (бесплатно)
1. Создать аккаунт на render.com
2. New → Background Worker
3. Build command: `pip install python-telegram-bot==20.7`
4. Start command: `python bot.py`
5. Добавить env variable BOT_TOKEN

### Вариант C: Запуск на своём компьютере
Просто оставить терминал открытым с запущенным `python bot.py`.
Бот работает пока компьютер включён.
